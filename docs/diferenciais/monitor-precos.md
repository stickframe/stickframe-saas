## Por que é um diferencial vs. Procore / Autodesk
- Procore não tem monitoramento de preços de insumos brasileiros
- Nenhum concorrente nacional integra variação de mercado direto no dashboard
- Permite ao diretor de obra reagir rapidamente a altas de material
- Único sistema que conecta **custo de obra em andamento** com **tendência de mercado** em tempo real

## Onde aparece no sistema
- Dashboard → card "Monitor de Preços" (visível imediatamente após login)
- Página Fornecedores → Monitor Preços (histórico completo)
- Alertas automáticos quando variação > threshold configurado

> "Enquanto o Procore exige integrações customizadas para dados de mercado,

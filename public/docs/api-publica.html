API Pública REST

Integre o StickFrame ao seu ERP, planilha ou sistema de terceiros com Bearer token — sem setup enterprise, sem meses de implantação.

REST / JSON
Bearer Token sf_live_xxx
Docs inline em /docs/api
📡

Endpoints Disponíveis

v1
Método Endpoint Descrição
GET /api/v1/obras Lista todas as obras da conta
GET /api/v1/obras/:id Detalhe completo de uma obra específica
GET /api/v1/clientes Lista clientes da conta
GET /api/v1/financeiro Resumo financeiro consolidado
POST /api/v1/obras Cria nova obra na conta
⚡

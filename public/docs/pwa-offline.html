Componentes Técnicos

Next.js
📋
manifest.json
Define nome, ícones, cor de tema e modo standalone. Responsável pela experiência de instalação no celular.
⚡
sw.js Service Worker
Cache de assets e dados para funcionamento offline. Servido pelo Next.js com estratégia stale-while-revalidate.
🖼️
Ícones PWA
Ícones em 192×192 e 512×512 px otimizados para iOS e Android. Compatível com splash screen nativa.
🔔
Push API
Notificações enviadas mesmo com o app fechado. Integrada ao Service Worker para entrega confiável.
